# Brent Crude Oil Price Analysis

This project analyzes the historical Brent Crude Oil prices from 1987 to 2022, using data sourced from Kaggle. The aim is to understand long-term price trends, yearly and decade-wise fluctuations, and major global events influencing oil prices

## Files

- `oil price analysis.ipynb"`: Jupyter Notebook with the full analysis and visualizations.
- `BrentOilPrices.csv`: The dataset used for analysis.

## Key Insights

- Highest price: $143.95 in 2008
- Lowest price: $9.1 in 1998
- Largest absolute price change: **$32.62** in **2022**
- Largest percentage price change: **68.30%** in **2021**
- Average price across all years: $48.42
